Nginx 模块

GeoIP Module

通过 nginx GeoIP 模块 (IP地址识别)实现限制某些国家地区访问网站和基于区域的负载均衡。

URL：http://nginx.org/en/docs/stream/ngx_stream_geoip_module.html

incubator-pagespeed-ngx

Google开发的 高性能传输、低延时，基于nginx的页面加速。

ngx_pagespeed可以自动将网页性能最佳做法应用于网页和相关资源（CSS，JavaScript，图片），从而加快您的网站速度并缩短网页加载时间，而无需修改现有内容或工作流程。功能包括：

  • 图像优化：剥离元数据，动态调整大小，重新压缩
  • CSS和JavaScript缩小，连接，内联和大纲
  • 小资源内联
  • 延迟图像和JavaScript加载
  • HTML重写
  • 缓存生存期延长
  • 和 更多

URL：http://ngxpagespeed.com/ngx_pagespeed_example/

Github：https://github.com/apache/incubator-pagespeed-ngx

ModSecurity

ModSecurity是一个开源的，跨平台的Web应用程序防火墙（WAF）引擎，用于Apache，IIS和Nginx，由Trustwave的SpiderLabs开发。它具有强大的基于事件的编程语言，可以防止对Web应用程序的一系列攻击，并允许HTTP流量监控，日志记录和实时分析……

核心功能：

  • HTTP保护 - 检测违反HTTP协议和本地定义的使用策略。
  • 实时黑名单查询 - 利用第三方IP信誉
  • 基于Web的恶意软件检测 - 通过检查Google安全浏览API来识别恶意网络内容。
  • HTTP拒绝服务保护 - 防止HTTP泛洪和慢速HTTP DoS攻击。
  • 常见Web攻击防护 - 检测常见Web应用程序安全攻击。
  • 自动化检测 - 检测机器人，爬虫，扫描仪和其他表面恶意活动。
  • 与AV扫描文件上传集成 - 检测通过Web应用程序上传的恶意文件。
  • 跟踪敏感数据 - 跟踪信用卡使用情况并阻止泄漏。
  • 特洛伊木马保护 - 检测对特洛伊木马的访问。
  • 识别应用程序缺陷 - 应用程序错误配置警报。
  • 错误检测和隐藏 - 伪装服务器发送的错误消息。

URL：https://www.modsecurity.org/

Github：https://github.com/SpiderLabs/ModSecurity

lua_nginx_moudel

将Lua的强大功能嵌入到Nginx HTTP服务器中。

URL：https://openresty.org/

Github：https://github.com/openresty/lua-nginx-module

VeryNginx

VeryNginx基于lua_nginx_module(openrestry)。它实现了高级防火墙（waf），访问统计和一些其他功能。它增强了Nginx的功能，并提供友好的Web界面。

功能

  • Nginx 运行状态分析：每秒请求数、响应时间、净流量、Tcp connectinn num
  • 自定义操作：自定义操作由两部分组成，Matcher和Action。Matcher用于测试请求是否符合规则，Action是您要运行的逻辑。
  • 请求统计：VeryNginx可以记录URI的请求，包括每个URI的这些数据：
    • 所有请求计数
    • 请求每个状态码的计数
    • 总字节数
    • 平均字节数
    • 总响应时间
    • 平均响应时间

Github：https://github.com/alexazhou/VeryNginx

ngx_http_google_filter_module

ngx_http_google_filter_module是一个过滤器模块，能够让谷歌镜像更便捷的部署。内建了正则表达式、URI locations和其他复杂的配置。原生nginx模块确保了更加高效地处理cookies, gstatic scoures和重定向。

Github：https://github.com/cuber/ngx_http_google_filter_module

ngx_lua_waf

ngx_lua_waf是一个基于lua-nginx-module(openresty)的web应用防火墙。

功能：

  • 防止sql注入，本地包含，部分溢出，fuzzing测试，xss,SSRF等web攻击
  • 防止svn/备份之类文件泄漏
  • 防止ApacheBench之类压力测试工具的攻击
  • 屏蔽常见的扫描黑客工具，扫描器
  • 屏蔽异常的网络请求
  • 屏蔽图片附件类目录php执行权限
  • 防止webshell上传

Github：https://github.com/loveshell/ngx_lua_waf

nginx-rtmp-module

基于NGINX的媒体流媒体服务器。

特性：

  • RTMP / HLS / MPEG-DASH直播
  • RTMP视频点播FLV / MP4，从本地文件系统或HTTP播放
  • 对分布式流媒体的流中继支持：推拉模型
  • 在多个FLV中记录流
  • H264 / AAC支持
  • 使用FFmpeg进行在线转码
  • HTTP回调（发布/播放/记录/更新等）
  • 在某些事件上运行外部程序（exec）
  • HTTP控制模块，用于录制音频/视频和丢弃客户端
  • 高级缓冲技术可将内存分配保持在最低水平，以实现更快的流式传输和更低的内存占用
  • 事实证明可以使用Wirecast，FMS，Wowza，JWPlayer，FlowPlayer，StrobeMediaPlayback，ffmpeg，avconv，rtmpdump，flvstreamer等等
  • XML / XSL中的统计信息，以机器和人类可读的形式
  • Linux的/ FreeBSD的/ MacOS的/视窗

URL：http://nginx-rtmp.blogspot.com/

Github：https://github.com/arut/nginx-rtmp-module
