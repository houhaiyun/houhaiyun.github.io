Centos6.9 安装详解

Linux基础Centos6.9 安装详解
  • Centos6.9 安装详解

CentOS（Community Enterprise Operating System，中文意思是：社区企业操作系统）是Linux发行版之一，它是来自于Red Hat Enterprise Linux依照开放源代码规定释出的源代码所编译而成。

虚拟机配置

开始安装

开启虚拟机后我们看到如下安装界面：

  1. 安装或更新系统

  2. 安装显示卡驱动

  3. 系统修复

  4. 从硬盘启动

  5. 内存测试

我们直接回车即可，即使你不动，默认也会在自动倒数结束后，开始安装。

进入下一个界面测试安装媒体

测试你的安装媒体是否损坏，按OK测试你的安装媒体（你所选的ISO文件），我们这里直接按SKIP跳过。

第一个界面

选右下角的“NEXT”，下一步。

然后就到了选语言，这里我们选英文。如图：

选择键盘

这个不要选错了，默认就可以。如图：

这里选择安装的存储设备

我想大家都是安装到本地硬盘吧，那就是上面这个了啦，第二个选线的意思是指定专用的存储设备（例如存储区域网络SAN），如图。选择：基本存储设备后，点击下一步。

弹出警告

会删除检测到的这个硬盘（204800MB）的所有数据，如果是多硬盘安装或升级安装的话，要小心注意了，全新安装的话，直接点击：是，如图。

这里是选择时区

我们选择：亚洲/中国/上海，如图。下一步。

这里是设置根账户（root）的密码

如图，如果你的密码过于简单，则会有另一个提示，我可以选择：无论如何都使用，如图：

设置主机名

设置网络

下面我们就到了分区：此处我们选择“创建自定义布局”

  • 使用所有空间
  • 替换现有的Linux系统
  • 缩小当前系统
  • 使用自由空间
  • 创建自定义布局

自定义分区：

挂载点 大小 分区名称
/ 50G /dev/sdba1
/boot 1G /dev/sda2
/app 50G /dev/sda3
swap 4G /dev/sda4

创建/boot分区

创建/分区

创建/app分区

创建交换分区

分区图

是否格式化

点击下一步，将修改写入磁盘

然后，到了选择安装Centos组件的时候

这一步是选择机子是做什么用的，如图，分别是：

  1. 桌面系统的安装
  2. 最小化桌面系统的安装
  3. 最小化安装
  4. 基本服务器的安装
  5. 数据库服务器的安装
  6. WEB网页服务器的安装
  7. 虚拟主机的安装
  8. 软件开发工作站的安装

选择一种，系统会自动给你安装上一些需要的软件，当然也可以选择底下的：现在自定义，来选择要安装的组件。

下一步后，就开始安装了。

安装过程则根据你选择的组件内容所用的时间而不同……如图。

安装完成后就可以重启系统了，按一下“reboot”。如图：

重启完成后，进入欢迎界面，点击“Forward”

是否同意，点击“Forward”

创建一个普通用户haiyun,也可以不创建

设置时间，如果时间不对可以重新调整

是否开启kdump

kdump是在系统崩溃、死锁或者死机的时候用来转储内存运行参数的一个工具和服务，此处就就不开启了。也可开启，根据需求自行选择

点击确认，需要重启

重启完成，如下图可以愉快的登陆系统了。
